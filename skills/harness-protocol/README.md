# Codebase Harness Protocol

A universal agent skill for helping coding agents work well inside real software repositories.

This skill turns OpenAI's harness engineering ideas into a reusable protocol: keep repository instructions small, make the codebase itself the source of truth, organize durable docs where agents can find them, and use tests, scripts, plans, and feedback loops to prevent drift.

## Why Use It

Coding agents struggle when project knowledge lives in chat threads, stale docs, or people's heads. Codebase Harness Protocol gives agents a shared way to rehydrate context, navigate repo documentation, verify changes, and improve the harness when they get stuck.

Use it for repos that need to work across Codex, Claude Code, Antigravity, Cursor, Copilot, Devin, and other agentic coding tools.
